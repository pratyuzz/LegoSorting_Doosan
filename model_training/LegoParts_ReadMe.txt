This README provides information about the LEGO pieces included in this project.


Naming Convention
The LEGO piece filenames follow the format: {PartID}_{Color}_{Orientation}

{PartID}: Refers to the part ID of the LEGO piece from the designated excel sheet.
{Color}: Refers to the color of the LEGO piece.
{Orientation}: Refers to the orientation of the LEGO piece.

Color Codes:
blk: Black
blu: Blue
beig: Beige
dg: Dark Gray
k: Khaki
lg: Light Gray
o: Orange
r: Red
w: White
y: Yellow
